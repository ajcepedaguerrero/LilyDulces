# Lily Dulces

Proyecto de pagina web de una dulceria.

## Opciones

- Muestra los productos
- Muestra video relacionado
- Muestra mapa de ubicacion
- Permite contacto mediante formulario

## Secciones
- Barra de navegacionde la pagina
- Productos
- Carrito de compras
- Video sobre los productos
- Mapa de ubicación de la empresa
- Formulario
- Reseñas

## Tecnologías utilizadas

- HTML
- CSS
